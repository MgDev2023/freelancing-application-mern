# Freelancing Application (MERN Stack)

A web application for connecting freelancers with clients, built using the MERN stack.

---

## What does it do?

This is a platform where:
- Clients can post jobs/projects
- Freelancers can apply for those jobs
- Both sides can communicate and manage work

---

## Tech used

- **MongoDB** — database
- **Express.js** — backend server
- **React.js** — frontend
- **Node.js** — runtime

This combination is called the MERN stack.

## How to run it locally

1. Go into the `freelancing-app 1` folder
2. Install dependencies:
```bash
npm install
```
3. Start the app:
```bash
npm start
```
